# AlmaTrack 🎓

> **Платформа для поддержания связи между вузом и студентами/выпускниками**

AlmaTrack — это комплексная система, созданная на хакатоне за 48 часов командой **«С.О.В.А.»**. Позволяет студентам вести карьерный дневник через Telegram-бот, а институту — видеть обезличенную аналитику по занятости выпускников.

---

## 🧩 Из чего состоит

| Компонент | Технология | Что делает |
|-----------|-----------|-----------|
| **Telegram Bot** | Aiogram 3 + Python | Интерфейс для студентов: профиль, занятость, геймификация |
| **REST API** | FastAPI + AsyncIO | Backend, соединяющий бота, сайт и базу данных |
| **Admin Web** | Flask + Jinja2 | Панель деканата: аналитика, модерация, мероприятия |
| **Database** | SQLAlchemy + SQLite | ORM-модели, миграции, seed-данные |

---

## ✨ Возможности

### Для студентов (Telegram Bot `@alma_track_bot`)
- **Онбординг за 2 минуты** — факультет, курс, год поступления
- **Карьерный дневник** — компания, должность, город, формат работы (офис/удалёнка/гибрид), уровень (`intern → lead`)
- **Геймификация** — XP-очки, 5 уровней карьеры, 8 уникальных бейджей, стрики
- **Уведомления** — push о мероприятиях института прямо в Telegram
- **Приватность** — три уровня, возможность скрыть конкретные поля

### Для института (Admin Panel)
- **Аналитика** — топ работодателей, распределение по уровням и факультетам
- **Карта городов** — где работают выпускники
- **Модерация** — проверка и одобрение анкет
- **Мероприятия** — создание событий с минимальным порогом уровня участника
- **Push-рассылка** — уведомление студентов из интерфейса прямо в Telegram

## 🏆 Система геймификации

```
🌱 Новичок    →    🚀 Стажёр    →    💼 Специалист    →    🔥 Профи    →    👑 Легенда
    0 XP            100 XP             300 XP              600 XP           1000 XP
```

**Как зарабатывать XP:**
- `+50 XP` — завершение онбординга
- `+40 XP` — добавление места работы
- `+30 XP` — обновление данных
- `+60 XP` — переход на следующий уровень
- `+15 XP` — регистрация на мероприятие
- `+25 XP` — стрик 7 дней подряд

## 🚀 Быстрый старт

### Требования
- Python 3.12+
- [uv](https://github.com/astral-sh/uv) (менеджер зависимостей)
- Telegram Bot Token (через [@BotFather](https://t.me/botfather))

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/xvin84/AlmaTrack.git
cd AlmaTrack

# 2. Установите зависимости
uv sync

# 3. Настройте .env
cp .env.example .env
# Отредактируйте .env: укажите BOT_TOKEN и другие переменные

# 4. Запустите все сервисы одной командой
uv run main.py
```

### Доступные сервисы после запуска

| Сервис | URL |
|--------|-----|
| Admin Panel | http://localhost:5000 |
| FastAPI Swagger | http://localhost:8000/docs |
| Telegram Bot | `@alma_track_bot` в Telegram |

## 🔐 Приватность и безопасность

Приватность — ключевой принцип AlmaTrack. Студент сам управляет видимостью данных:

| Уровень | Что видит институт |
|---------|-------------------|
| `0` — Скрытый | Только обезличенная статистика |
| `1` — Обычный (по умолчанию) | Факультет, уровень, сфера занятости |
| `2` — Открытый | Полный профиль с компанией |

Каждое поле занятости (`is_hidden`) можно скрыть отдельно.
