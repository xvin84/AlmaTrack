# AlmaTrack - Платформа для работы с выпускниками

AlmaTrack — это комплексная система отслеживания карьерного пути и поддержания связи с выпускниками вуза.

## 🚀 Особенности
- **Telegram Bot (@alma_track_bot)**: Удобный интерфейс для онбординга, обновления профиля и Геймификации (опыт, уровни, бейджи).
- **FastAPI Backend (API)**: Мощный и быстрый асинхронный бэкенд, соединяющий все компоненты (Сайт, Бот, Базу).
- **Web Dashboard (Flask)**: Панель администратора для проверки анкет, модерирования участников, просмотра аналитики и создания мероприятий.
- **Push-уведомления**: Асинхронные уведомления в Telegram прямо из админ-панели (при создании / редактировании мероприятий).
- **Дашборд-Аналитика**: Отслеживание статистики участников по уровням (Стажёр, Junior, Senior и др.), факультетам и годам.

## 🛠 Установка и Запуск

Проект поставляется с единым скриптом инициализации:
```bash
# 1. Загрузите зависимости
uv sync

# 2. Запустите все сервисы одним скриптом (Web, API, Bot)
uv run main.py
```

Системы:
- **Web Администраторская**: `http://localhost:5000`
- **FastAPI Swagger**: `http://localhost:8000/docs`

## 🏆 Система Геймификации
AlmaTrack поддерживает вовлечение через награды:
- 5 уровней квалификации (от 🌱 *Новичок* до 👑 *Легенда*)
- XP за заполнение профиля, обновление работ и посещение мероприятий.
- 8 уникальных бейджей, вручаемых за специфические условия (например, *Первый шаг*, *Удалёнщик*, *Выпускник*).

## 📅 Мероприятия
Администраторы могут создавать события, указывая минимальный порог «Уровня/Опыта» участника для регистрации напрямую через Telegram-бота. Рассылка работает в фоне (Asyncio Tasks), чтобы не блокировать интерфейс сайта.
